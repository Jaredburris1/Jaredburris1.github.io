Twitter Script
Python Data Analysis

I wrote this Python Data Analysis twitter script to count the total amount of tweets from a specific hashtag from a list of users while also pulling each one of those tweets and storing them in a .json file. This .json file can be parsed/formatted and stored within Pandas for easy analysis with graphs/etc. I was commissioned with this assignment by an NFT project to promote members to tweet for the said project, this would allow for constant tracking with ease, expectation of rank and potential gain in investments/collections. My plan is to host this script on a website to keep my intellectual property secure and implement a database of all the members for simple additions and removal of members. This website will also have an easy way for the admins of the project to log in, click a button to get the daily totals for analysis.

Open When...
iOS Application

I am currently in constant development of this iOS application; I consider it my hobby I really enjoy doing in my spare time and I think it is an amazing idea. The project is about an Open When Letter application that can be used by loved ones around the world, the idea is to have accounts for each user and each one of these users can add friends/family on their accounts, after the friend request is accepted either party can now send a letter to the other. The letters themselves are going to be time locked based on the sender's time allotment and when it comes that time the letter can be opened and read by the recipient, I am hosting a firebase database to hold each account with their respective letters and even have the potential of putting extra items inside each letter, for example a bit of money from a grandparent. Another good example would be Military members sending time opened letters for when they get out of bootcamp. I would say I am working out the front-end of the application currently with firebase already set up, I just need to work out the kinks in the front-end and then finish development up in the back-end and it will be ready to send to the Apple Developers!
